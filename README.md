# Workspace Manager - Project and environment management

**Package**: `@experimental/workspace`
**Version**: 0.1.0
**Repository**: `pkg-experimental-workspace`

## Overview

Workspace Manager - Project and environment management

## Installation

### Prerequisites

No external dependencies required.

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @experimental/workspace

# Or install from local source
cd ~/.pkgs
stow workspace
```

## Usage

### Experimental

**⚠️ Warning**: This package is experimental and may change significantly.

Workspace Manager - Project and environment management

## Configuration

Configuration files are typically stored in:
- `~/.config/workspace/` - User configuration
- `~/.local/share/workspace/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @experimental/workspace

# Or manual unstow
cd ~/.pkgs
stow -D workspace
```
